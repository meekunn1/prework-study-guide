Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it. (replace name with "branch-name")
  • checkout = move the working branch to a new branch
  • -b = create a new branch
  • git pull origin main = get latest version of the project on your local computer
  • git add -A = Commit all files in the working branch.
  • git commit -m "insert message here" = associate message to commit.
